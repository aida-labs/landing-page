    Introduction

    Welcome to the Aida Labs CLI documentation! Here we'll go into detail all about how to start using our hosted Spark clusters and run your Spark applications on them.

    You can download our CLI through brew.

    Brew install

    To install the CLI, run this command:

    brew tap aida-labs/aida

brew install aida

    After registering for an account here on the website, you can get started with the Aida CLI with a brew install command.

    Login and Run

    Login

    aida auth:login

    You're all set up! You can now run any Spark python file in production on a Cluster with aida run spark_app.py!

    aida run spark_app.py

    If you happened to get any error messages, then reach us on our community Discord and we can fix that right away.
